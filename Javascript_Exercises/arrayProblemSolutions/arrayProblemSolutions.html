Array Problem Set Solutions

  • printReverse() - takes an array as an argument and prints the elements in an array in reverse order
  • isUniform() - takes an array as an argument and returns true if all elements in the array are identical
  • sumArray() - accepts an array of numbers as an argument and returns the sum of all numbers in the array
  • max() - accepts an array of numbers as an argument and returns the maximum number in the array
