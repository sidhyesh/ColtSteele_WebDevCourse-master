  • val()

    • Checks the value of inputs ($("input").val() can check contents of a text box

  • attr()

    • $("img").attr("src", "https://s-media-cache-ak0.pinimg.com/564x/0f/16/4b/0f164b264291617ff04b9f6dcde974f6.jpg") change picture in the bottom

  • html()

    • Can be used to look at what's inside a pair of html tags, equivalent to .innerHTML in Vanilla JS

  • addClass()

    • adds a passed class to html tags (.addClass("correct") adds "correct" class

  • removeClass()

    • removes the passed class from html tags

  • toggleClass()

    • adds if passed class isn't applied, removes if it is. $("li").toggleClass("done")
